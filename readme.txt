--------------------------- INTRUÇÕES DE COMPILAÇÃO E USO ---------------------------

Para rodar o programa, abra o arquivo ScytlAppDefinitivo.xcworkspaces. Aperte o botão de play e aguarde o simulador abrir.

O programa é baseado em 3 telas, sendo estas:
 - Página inicial: Lista todas as tarefas, sendo possível clicar em qualquer uma delas para editar, deletar ou apenas exibir os detalhes. Possui também o botão "Create New Schedule", utilizado para criar uma nova tarefa. Caso o servidor demore a responder, aperte o botão de refresh para atualizar os dados

 - Página de editar ou deletar tarefa: Após selecionar uma tarefa na página inicial, essa página irá abrir e mostrará todos os detalhes da tarefa selecionada, podendo editar ou deletar a tarefa. Antes de deletar aparecerá uma mensagem de confirmação de operação, para evitar que o usuário delete sem querer.

 - Página de criar nova tarefa: Após clicar no botão de criar nova tarefa, essa página abrirá. O usuário poderá editar da forma que quiser e no final clicar no botão de criar tarefa.
